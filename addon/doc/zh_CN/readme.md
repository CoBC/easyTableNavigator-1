# Easy Table Navigator-简单表格导航插件 #

* 作者： Corentin Bacqué-Cazenave, Joseph Lee
* 下载 [稳定版][1]
* 下载 [开发板][2]
* NVDA兼容版本： 2019.3 到 2021.1

此插件添加了一个图层命令来使用光标键来导航表单元格。

目前支持的表格有：

* 浏览模式（Internet Explorer，Firefox等）。
* Microsoft Word。

## 快捷键

* 切换表格导航器图层（未指定）。
